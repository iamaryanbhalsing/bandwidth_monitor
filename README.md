# 📡 Bandwidth Monitor

A lightweight Python utility to **track your internet data usage** in real-time during your current online session.

---

## ✨ Features

- **Real-time Monitoring**: Tracks upload & download speeds
- **Session Stats**: Total data used in current session
- **Hourly Analysis**: Identifies peak usage hours
- **Visual Report**: Generates beautiful bar graph
- **CSV Logging**: Saves history for long-term analysis
- **Cross-Platform**: Works on Windows 11, Linux & macOS

## 🛠️ Installation

### 1. Clone or Download
```bash
git clone https://github.com/yourusername/bandwidth-monitor.git
cd bandwidth-monitor
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. How to Use
```bash
python bandwidth_monitor.py
```

Run the script
Keep it running while using the internet
Press Ctrl + C to stop and generate report
